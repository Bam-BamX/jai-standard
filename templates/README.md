# templates/

Canonical templates for handoffs, dev prompts, review reports, security reports, etc. Project `/jai/templates/` may carry rare overrides; otherwise projects use canonical versions by reference.

**Empty at v1.0** — lazy migration. The 4 templates currently live in Trend Scanner v1's `jai/templates/` (per the project-level implementation plan):
- `dev_prompt_template.md`
- `handoff_template.md`
- `review_report_template.md`
- `security_report_template.md`

Each will be lifted into a canonical version here when first amended. Until then, projects use the implementation-plan versions and the JAI Standard v1.0 inherits them by reference.

Adding or amending a canonical template is a MINOR version bump and follows the [Enhancement Intake workflow](../JAI_GOVERNANCE_STACK.md#4--enhancement-workflow).
